# Python Tkinter学习记录



## 1.创建新窗口

导入tkinter(系统自带,如果没有,就要`pip install tkinter`),以下tkinter简称tk

首先输入我们的第一行命令

```python
import tkinter as tk #导入tkinter并将其命名为tk
```

紧接着写

```python
window=tk.Tk() #这里的window是变量名,可以自定义,后面的第一个tk表示刚才命名的`,作用是为了定义

window.mainloop() #窗口主循环
```

这样就可以看见我们的新窗口了

默认的标题为tk,如果不喜欢,可以参考01.py,在定义和窗口主循环之间加入window.title('随便写一个标题')



## 2.改变默认窗口大小和获取用户屏幕分辨率

将01.py文件复制一份,改名为02.py,**注意:后面为了方便,我就起名a1**

在上次的基础上,我们在主循环钱加入,**注意中间乘号是小写的x**

```python
a1.geometry('400x500')
```

这个是窗口大小,如果要想要固定开启位置,就加入

```python
a1.geometry('400x500+200+300')
```

但是我们每个人的屏幕分辨率不一样,比如1920x1080,1366x768,所以我们要获取每个人电脑屏幕的分辨率,从而更好的适配每一台电脑

获取用户分辨率:

```python
a2=a1.maxsize()

print(a2)
```

我们可以使用这个k,g分别对应的是屏幕的x轴,y轴,这里代表的是大小各位屏幕的一半

```python
k,g=a2

a1.geometry(f'{int(k/2)}x{int(g/2)}')
```



## 3.窗口锁定大小与图标

不难发现我们之前的窗口经常会被托拉拽,我们为了避免用户乱拉窗口,导致各组件无法正常运行,我们就要加入

```python
a1.resizable(False,False)  #前面一个是x,后面一个是y
```

resizable英文是可改变的,True代表可以改变,False代表不可以改变大小

接下来设置窗口图标,图标可以在网上找,然后找转换器转换下格式

```
#设置窗口图标,用相对路径更好*

a1.iconbitmap('logo.ico')
```
